Database Reference¶

General¶

  • Date / time are stored as strings in the format “yyyy-mm-ddTHH:MM:SS”, e.g., “2017-12-31T01:30:45”.

Transformers¶

  • Transformers must have same number of phases per winding.
  • Transformer must have exactly two windings.
